# 用户注册

## URL

/base/frame/register

## 支持格式

JSON

## HTTP请求方式

POST

## 是否需要登录

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
username     | 是   | string  | 账号
password     | 是   | string  | 密码 sha1

## 注意事项

## 返回结果

### 成功

```json
{}
```

### 错误

```json
{
  "code": 20000,
  "error": "用户不存在"
}
```

### HTTP状态

- 401 Unauthorized - [*]：表示用户没有权限（令牌、用户名、密码错误）。
- 403 Forbidden - [*] 表示用户得到授权（与401错误相对），但是访问是被禁止的。
