# 权限组删除

## URL

/base/permissions/:id

## 支持格式

JSON

## HTTP请求方式

DELETE

## 是否需要登录

是

如果request中的认证信息不正确，要返回状态码 401，即NOT AUTHORIZED

## 请求参数

字段 | 必选 | 类型及范围 | 说明
----|------|----------|-------------
id      | 是   | number  | 数据id

## 注意事项

### HTTP状态

- 204 NO CONTENT - [DELETE]：删除数据成功。

## 返回结果

不需要内容
